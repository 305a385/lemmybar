### lemmybar - redyt for lemmy

inspired by [redyt](https://github.com/Bugswriter/redyt)

returns images from a lemmy community

## dependencies
- imv (showing images)
- htmlq (parsing html)

## examples

```
$ ./lemmybar lemmy.ml memes
```
